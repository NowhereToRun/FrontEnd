布局要点

  1. 侧边栏和主内容之外使用wrapper， display:inline-block
  2. 侧边栏利用float:left到wrapper的边框上实现两栏布局
  3. main在wrapper的内部，所以不需要再为main设置margin-left或者overflow:hidden防止main的高度比left高时的内容溢出到左侧下方
  4. 等高布局一般都是为了显示背景颜色,利用侧边栏和主内容外层的wrapper实现等高
  5. 为了占满全屏 给wrapper设置了min-height 100%

wrapper display:inline-block的原因

  1. 创建BFC，清除内部浮动，让float:left的侧边栏能撑开wrapper的高度
  2. 不能使用overflow:hidden或者auto，这样会使侧边栏无法显示

此外wrapper需注意

  1. 因为是inline-block，所以不会自动占满整行。如果内部没有能占满整行的元素就会有显示问题（一般不会出现）。为了显示效果需设置width:100%和box-sizing:border-box
  2. 此外一种方法是直接给body设置background-color，这个color的颜色就直接显示在了main的内容的位置。从视觉上不会再有问题
        html, body {
            margin: 0;
            padding: 0;
            height: 100%;
        }

        #wrapper {
            display: inline-block;
            border-left: 200px solid #d4c376;
            background: #ffe3a6;
            /*vertical-align: bottom;*/
            min-height: 100%;
            width: 100%;
            box-sizing: border-box;
        }

        #sidebar {
            float: left;
            width: 200px;
            margin-left: -200px;
        }
                    
                        <div id="wrapper">
                            <div id="sidebar"></div>
                            <div id="main"></div>
                        </div>
