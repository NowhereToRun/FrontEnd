布局要点

  1. 利用超大padding和-margin实现等高布局，这样等高的缺点是如果为他们设置下边框将会无法显示
  2. 左侧float:left
  3. 右侧overflow:hidden或者margin-left:200px防止内容溢出，推荐使用overflow:hidden因为同时可以创建BFC防止边界塌陷
  4. 注意右侧不需要设置宽度
        html,body {
            margin: 0;
            height: 100%;
        }

        .container {
            overflow: hidden;
            min-height: 100%;
        }

        .left {
            float: left;
            width: 200px;
            background-color: lightskyblue;
            margin-bottom: -200001px;
            padding-bottom: 200001px;
        }

        .main {
            background-color: greenyellow;
            overflow: hidden;
            margin-bottom: -200001px;
            padding-bottom: 200001px;
        }
                    
                        <div class="container">
                            <div class="left"></div>
                            <div class="main"></div>
                        </div>
