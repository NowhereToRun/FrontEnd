利用两层wrapper实现视觉上的等高（背景颜色）

  1. 最外层显示的是左边栏的背景颜色
  2. 第二层显示的是主要内容的背景颜色
  3. 为了能把左边栏的颜色空出，第二层要相对定位，向右偏移左边栏的宽度。也可以直接margin-left:左边栏的宽度
  4. 剩下的布局就和利用负边距布局一样，只是为了恢复到正常位置（因为第二层右移了），left和content需要向左偏移同样的宽度
  5. contentInner不需要改变
  6. 最外层需要overflow:hidden来消除溢出的宽度
  7. container1 和 container float:left的原因是为了清除内部浮动。 其中内层（container）不能使用overflow:hidden代替，因为左边栏要利用负边距移动出去
        html, body {
            padding: 0;
            margin: 0;
        }

        #container1 {
            float: left;
            width: 100%;
            overflow: hidden;
            /*position: relative;*/
            background-color: #dbddbb;
        }

        #container {
            background-color: orange;
            width: 100%;
            float: left;
            margin-left:220px;
/*            position: relative;
            left: 220px; */
        }

        #left {
            float: left;
            margin-right: -100%;
            margin-left: -220px; /* 宽度大小等与边栏宽度大小*/
            width: 220px;
        }

        #content {
            float: left;
            width: 100%;
            margin-left: -220px; /* 宽度大小等与边栏宽度大小*/
        }

        #contentInner {
            margin-left: 220px; /* 宽度大小等与边栏宽度大小*/
            overflow: hidden;
        }

    <div id="container">
        <div id="left">
            Left Sidebar
        </div>
        <div id="content">
            <div id="contentInner">
            </div>
        </div>
    </div>
</div>
